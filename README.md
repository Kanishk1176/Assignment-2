# Assignment-2
This repository contains the solutions for **Task 1** and **Task 2** of the Assignment-2.

## Files in this repository
- **Assignment2_task1.py** – Checks whether the number entered is even or odd.  
- **Assignment2_task2.py** – Calculates the sum of integers from 1 to 50

##Description

These programs demonstrate basic Python concepts such as:
-Variables
-Input and output
-Conditional statements
-Loops
-Basic calculations
Each script follows the assignment instructions and produces the required output.
